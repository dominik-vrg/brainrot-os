# Brainrot OS
Stardance - web OS challange

This is my web-based OS project for the Stardance Challange.

------------------------------

My OS is called Brainrot OS.
# **BOOT**
It features a meme/brainrot themed boot animation when opening the webpage.

# **DESKTOP**
After the system "boots", there are three applications on the desktop, but also a working clock and a date display. !!The setting feature doesn't work yet!! All of the applications open a draggable window.

# **APPS**
  - brainrot.exe - just opens a draggable window with some text
  - notes - a simple, but functional note taking app
  - do not open - this is a minigame, when you open the app, it opens a window with a big red button that says do not click, if you click it, multiple error windows start popping up and you have to close all of them

------------------------------

I've used Claude AI to help with design (colors/theme), help with scripting, because I am new to JavaScript and text generation for the errors minigame and boot screen.
